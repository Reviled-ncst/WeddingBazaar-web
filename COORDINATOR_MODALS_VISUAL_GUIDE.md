### **Opening Modals**
```
1. User clicks action button (View/Edit/Delete/Add)
2. Modal fades in (300ms animation)
3. Backdrop darkens (50% opacity)
4. Focus moves to modal
5. Scroll locked on background
6. ESC key enabled to close
7. Click outside to close (optional)
```

### **Form Submission**
```
1. User fills/modifies form
2. Client-side validation on blur
3. Submit button enabled when valid
4. Click submit button
5. Button shows loading spinner
6. API call made
7. Success: Close modal + refresh data
8. Error: Show error message in modal
```

### **Closing Modals**
```
1. Click âŒ close button
2. OR click backdrop
3. OR press ESC key
4. Modal fades out (300ms animation)
5. Backdrop lightens
6. Focus returns to button that opened it
7. Background scroll restored
```
